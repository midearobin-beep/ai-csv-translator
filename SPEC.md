# AI CSV 翻译工具规范文档

## 1. 项目概述

- **项目名称**: AI CSV Translator
- **项目类型**: 网页应用 (Single Page Application)
- **核心功能**: 基于 AI 大模型批量翻译 CSV 文件内容
- **目标用户**: 需要批量翻译 CSV 数据的用户
- **技术栈**: 纯 HTML + CSS + JavaScript，无需后端

## 2. 文件结构

```
csv-translator/
├── index.html    # 主页面
├── styles.css   # 样式文件
├── app.js       # 应用程序逻辑
└── SPEC.md      # 本规范文档
```

## 3. UI/UX 规范

### 3.1 布局结构

- Header: "AI CSV 翻译工具"
- 文件上传区域
- 源语言选择
- 目标语言选择 (24种语言多选)
- 进阶功能 (字符长度控制)
- 预估完成时间显示
- 翻译按钮 + 进度条
- 数据预览表格
- 翻译结果表格 + 下载按钮
- API 设置 (底部，可折叠)

### 3.2 配色方案

- 主色: #2563EB (蓝色)
- 成功: #10B981 (绿色)
- 警告: #F59E0B (橙色)
- 危险: #EF4444 (红色)
- 背景: #F8FAFC
- 超长高亮: #FEF3C7

### 3.3 响应式断点

- 移动端: < 768px
- 桌面: >= 768px

### 4.2 API 设置

- DeepSeek 格式 (默认)
- OpenAI 兼容格式 (自定义 Base URL)

### 4.3 进阶功能

- 字符长度控制 (默认关闭)
- 倍率选择: 1.0x ~ 2.0x (默认 1.2x)
- 超长单元格高亮显示

### 4.4 预估时间计算

- 公式: 总请求数 × 1.5秒/请求
- 显示: "共 X 行 × Y 种语言 = Z 次请求，预计 M 分钟"

## 5. 用户交互流程

1. 上传 CSV 文件 → 显示文件信息 + 数据预览
2. 选择源语言
3. 选择目标语言 → 显示预估时间
4. (可选) 开启进阶功能，设置倍率
5. 点击"开始翻译"
6. 锁定所有配置项
7. 显示进度条 + 剩余时间
8. 翻译完成 → 显示结果 + 超长统计
9. 下载翻译后的 CSV
10. 可重新上传新文件

## 6. 字符计算规则

- 中文/日文/韩文: 2 字符
- 其他字符: 1 字符
