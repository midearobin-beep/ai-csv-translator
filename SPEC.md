# AI CSV 翻译工具规范文档

## 1. 项目概述

- **项目名称**: AI Table Translator
- **项目类型**: 网页应用 (Single Page Application)
- **核心功能**: 基于 AI 大模型批量翻译 CSV 文件内容
- **目标用户**: 需要批量翻译 CSV 数据的用户
- **技术栈**: 纯 HTML + CSS + JavaScript，无需后端

## 2. 文件结构

```
csv-translator/
├── index.html    # 主页面
├── styles.css   # 样式文件
├── app.js       # 应用程序逻辑
└── SPEC.md      # 本规范文档
```

## 3. UI/UX 规范

### 3.1 布局结构

- Header: "AI CSV 翻译工具"
- 翻译模式选择 (多语言列表 / 单语言替换)
- 文件上传区域
- 源语言选择
- 目标语言选择 (根据模式显示)
  - 多语言列表模式: 24种语言多选网格
  - 单语言替换模式: 单一下拉选择
- 进阶功能 (字符长度控制)
- 预估完成时间显示
- 翻译按钮 + 进度条
- 数据预览表格
- 翻译结果表格 + 下载按钮

## 4. 功能规范

### 4.1 翻译模式

- **多语言列表模式**: 翻译 CSV 第一列，生成多列目标语言（原有功能）
- **单语言替换模式**: 翻译所有单元格，原地替换原位置内容（新增）

### 4.3 API 设置

- DeepSeek 格式 (默认)
- OpenAI 兼容格式 (自定义 Base URL)

### 4.4 进阶功能

- 字符长度控制 (默认关闭)
- 倍率选择: 1.0x ~ 2.0x (默认 1.2x)
- 超长单元格高亮显示

### 4.5 预估时间计算

- 多语言列表模式: 总请求数 = 行数 × 目标语言数
- 单语言替换模式: 总请求数 = 行数 × 列数
- 每次请求预估耗时: 1.5秒

## 5. 用户交互流程

1. 上传 CSV 文件 → 显示文件信息 + 数据预览
2. 选择翻译模式
3. 选择源语言
4. 选择目标语言 → 显示预估时间
5. (可选) 开启进阶功能，设置倍率
6. 点击"开始翻译"
7. 锁定所有配置项
8. 显示进度条 + 剩余时间
9. 翻译完成 → 显示结果 + 超长统计
10. 下载翻译后的 CSV
11. 可重新上传新文件

## 6. 字符计算规则

- 中文/日文/韩文: 2 字符
- 其他字符: 1 字符
